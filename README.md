# Translate_Python

### google_translate文件 

#### config.py </br>

该文件是数据库的而配置文件，以及爬虫的随机请求头

#### HandleJs.py </br>

该文件主要是获取google翻译url中的tk字段

#### translate.py </br>

该文件是一些主要的函数，实现数据库的查询，语句的自动翻译，自动切片（google翻译最大的字符限制为5000）
